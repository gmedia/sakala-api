# Governance

Sakala API adalah bagian dari project deployment open-source Sakala yang dikembangkan melalui roadmap, dokumentasi, issue, dan kontribusi terbuka.

## Project Stewardship

Pada fase MVP, keputusan teknis dijaga oleh maintainer awal Sakala dengan dukungan GMEDIA sebagai founding sponsor dan initial infrastructure supporter.

## Sponsor Role

Sponsor dapat mendukung infrastruktur, domain, resource pengembangan, dokumentasi, dan kegiatan komunitas. Dukungan sponsor tidak otomatis memberikan kendali atas keputusan teknis, prioritas roadmap, perubahan license, atau hak contributor.

## Technical Decisions

Keputusan dibuat melalui issue, pull request, discussion, review maintainer, dan dokumentasi arsitektur. Perubahan security boundary, license, governance, dan kontrak lintas repository memerlukan review maintainer.

## Future Governance

Setelah MVP stabil dan kontribusi komunitas meluas, Sakala dapat berpindah ke organisasi komunitas khusus dan mengadopsi model maintainer yang lebih formal.
